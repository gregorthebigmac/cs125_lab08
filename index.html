Valid Commands: What it does:
PUSH stringAdds string to the queue. Nothing needs to be displayed. The string is every this after the space after PUSH until the end of the line.
SHOWdisplays the contents of entire queue across the screen as a table with one row.
POPremoves the first element from the queue. Nothing needs to be displayed. POPping from a empty queue will cause an error message and the program will shut down.
FRONTDisplays the first element on the queue only. Nothing in the queue changes. Calling FRONT on an empty queue will cause an error message and the program will shut down.
Any other commands will display that an invalid command was entered, the command will be displayed and exit the program.
